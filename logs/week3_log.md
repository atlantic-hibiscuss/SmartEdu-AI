# Week 3 (Days 19–30) — LangChain & Conversational Memory

**Focus:** Learn core LangChain concepts; add conversation memory; make a CLI tutor.  
**Deliverables:** CLI chatbot with either full-history memory or windowed memory.

## Step-by-step

1) **Concept refresh**
- LLM, Prompt, Chain, Memory: understand how they connect.

2) **Run the CLI with full history**
```bash
python src/week3/cli_chatbot.py
```

3) **Run the CLI with windowed memory (last k turns)**
```bash
python src/week3/cli_chatbot.py 5
```

4) **Try conversations**
- Ask follow-ups; observe how it remembers prior messages.
