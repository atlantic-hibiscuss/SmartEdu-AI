# Week 2 (Days 9–18) — LLM Interaction & Core Chatbot Functionality

**Focus:** Text preprocessing, pre-trained LLM interaction, and tutor-style prompts.  
**Deliverables:** Tokenization notebook/script; text-generation demo; initial tutor prompt.

## Step-by-step

1) **Set up environment**
```bash
python -m venv .venv
# Windows: .venv\Scripts\activate
# macOS/Linux:
source .venv/bin/activate
pip install -r requirements.txt
```

2) **Tokenization practice**
- File: `src/week2/tokenize_demo.py`
- Run:
```bash
python src/week2/tokenize_demo.py
```

3) **Text generation with a small model**
- File: `src/week2/generation_demo.py`
- Run:
```bash
python src/week2/generation_demo.py
```

4) **Create a tutor prompt**
- File: `src/week2/prompts/tutor_system_prompt.txt`
- Edit the prompt to match your tone and domain.
